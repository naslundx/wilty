Changelog - v0.2

✨ New Features

  • Personalized Secrets: You can now specify if a secret statement you submit should only apply to you! If you check "Only apply to me", you will be guaranteed to be the storyteller when it gets drawn.
  • Storyteller Skip: If a drawn statement leaves you stumped, you can now skip it! A "Skip & Get New Prompt" button appears during the initial 5-second preparation delay.
  • Admin Global Directory: Admins can now promote user-submitted statements from a specific room into the global directory directly from the dashboard.
